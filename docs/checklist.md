# Checklist

The Checklist tracks the pre-event readiness status of every team. It's useful for RIs, FTAs, and FTAAs to see at a glance which teams still need to complete inspection steps.

Navigate to it at **ftabuddy.com/checklist** or through the menu.

---

## Tracked Statuses

Each team has five checkboxes:

| Status                | What it means                                               |
| --------------------- | ----------------------------------------------------------- |
| **Present**           | Team has checked in and is at the event                     |
| **Weighed**           | Robot has been weighed by inspection                        |
| **Inspected**         | Robot has passed full inspection                            |
| **Radio Programmed**  | Robot's radio has been flashed with the correct team number |
| **Connection Tested** | Robot has passed a DS connection test                       |

---

## Updating the Checklist

Tap any checkbox to toggle it. Changes are saved immediately and visible to all event volunteers in real time.

---

## Nexus Integration

If [Nexus](./integrations#nexus) is configured for the event, **Connection Tested** results are automatically imported from FRC Nexus whenever a team passes their connection check. You can still manually override any status.

---

## Filtering Teams

Use the search/filter bar to find a specific team by number or name. This is helpful when the pit is crowded and you need to quickly look up a team's status.

---

## Inspection Alerts on Field Monitor

If a team that hasn't been inspected is on the field, a 🔍 icon appears on their station in the Field Monitor (requires **Inspection Alerts** enabled in [Settings](./settings)).
