# FTA Buddy Documentation

FTA Buddy is a companion app for FIRST Robotics Competition (FRC) event volunteers. It provides live field monitoring, team support notes, match log analysis, and event management tools - all without overloading the FMS network.

## User Guides

- [Getting Started](./getting-started) - accounts, roles, joining events, installing the app
- [Hosting an Event](./hosting-an-event) - browser extension setup, event creation, event settings

### Features

- [Field Monitor](./field-monitor/) - live robot status for all 6 stations
- [Notepad](./notepad/) - team support notes and tickets
- [Match Logs](./match-logs/) - reviewing and sharing DS connection logs
- [Checklist](./checklist) - team inspection and pre-match status tracking
- [Flashcards](./flashcards) - quick-reference cards for the field
- [References](./references) - status lights, wiring diagrams, manuals, and more
- [Settings](./settings) - customizing alerts, appearance, and notifications
- [Integrations](./integrations) - FTA App sync, Slack, Nexus, and kiosk modes

## Developer Reference

- [Public API](./api/) - REST endpoints for match logs and cycle times
