# Team Ticket Submissions

Teams can submit their own support tickets without needing an FTA Buddy account. This is useful for allowing pit crew or drive teams to report issues directly.

---

## Getting the Submission Link

The public submission URL for an event follows this format:

```
https://ftabuddy.com/notepad/submit/[event-code]
```

Replace `[event-code]` with your event's code (e.g. `2026mitry`).

You can share this link as a QR code posted in the pit area, or distribute it to teams directly.

---

## What Teams See

When a team opens the submission link they see a simple form:

- **Team number** field
- **Description** of the issue
- **Submit** button

No login is required. Submissions appear in the Notepad feed immediately and are visible to all event volunteers.

---

## Handling Submitted Tickets

Submitted tickets appear as **Team Issue** notes in the Feed View, marked to indicate they were team-submitted. Volunteers can add categories, assign them, and resolve them the same way as any other note.
