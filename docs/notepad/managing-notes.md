# Managing Notes

## Note Status

Every note has one of three statuses:

| Status             | Meaning                                               |
| ------------------ | ----------------------------------------------------- |
| **Open**           | Issue is unresolved and needs attention               |
| **Resolved**       | Issue has been addressed                              |
| **Not Applicable** | Note is no longer relevant (e.g. team didn't show up) |

The Feed View defaults to showing only **Open** notes. Use the status filter to view resolved or all notes.

---

## Resolving a Note

Open the note and tap **Resolve**. You can add a resolution comment before saving.

---

## Editing a Note

Open the note and tap the **Edit** (pencil) icon to change the description, categories, or type.

---

## Deleting a Note

Open the note and tap the **Delete** (trash) icon. This permanently removes the note.

---

## Assigning a Note

Notes can be assigned to a specific volunteer to track who is handling an issue:

1. Open the note.
2. Tap **Assign**.
3. Select a volunteer from the list (populated from event users).

The assigned person receives a notification (if they have notifications enabled and are subscribed to assignment alerts).

---

## Commenting on a Note

You can leave messages on a note to track progress or communicate with other volunteers:

1. Open the note.
2. Type in the **message** box at the bottom.
3. Tap **Send**.

Comments appear in chronological order on the note.

---

## Viewing a Team's History

To see all notes ever created for a specific team:

- From **Feed View**, tap a note for that team, then tap the team number at the top.
- From the **Field Monitor**, tap the team number cell and then navigate to their note history.
- Navigate directly to `/notepad/team/[team-number]`.

The team history page shows all notes for that team, including resolved ones, and lists which matches they've played since each note was created.

---

## FTA App Sync

If the [FTA App integration](../integrations#fta-app-sync) is configured, notes are synced bi-directionally with the official FMS FTA App in real time. Notes created in FTA Buddy appear in the FTA App and vice versa.

---

## Notifications for Notes

You can subscribe to notifications for:

- **New notes** created at the event
- **Notes you follow** being updated
- **Notes assigned to you** being updated

Notification preferences are in [Settings](../settings#notifications).
