# References

The References section is a library of quick-access documentation for FRC hardware and software. It's available to all users without needing to log in or join an event.

Navigate to **ftabuddy.com/references**.

---

## Status Lights

**ftabuddy.com/references/statuslights**

An interactive guide to the LED status indicators on FRC control system components. For each component, you can look up what a specific blink pattern means.

Covers:

- **Radio** (OpenMesh OM5P-AN) - power, boot status, link indicators
- **roboRIO** - status LEDs for boot, code, communication
- **Spark MAX / Spark Flex** (REV Robotics) - motor controller status
- **Power Distribution Hub (PDH)** and **Power Distribution Panel (PDP)**
- **Pneumatics Hub / Pneumatics Control Module**
- **CTRE devices** (Talon FX, Talon SRX, Victor SPX, CANcoder, Pigeon IMU)

## FiM-Specific References

If **FiM Specific Field Manuals** is enabled in [Settings](./settings), an additional link to FiM (FIRST in Michigan) specific documentation appears on the References page. This is on by default and can be turned off for non-FiM events.
